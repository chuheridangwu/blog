# 持仓

## 中长期交易

有不少读者问过我，做中期波段趋势的时候，都知道，应该**前重后轻**。

比如你总共的资金20份。

所谓买入，第一笔下四份，第二笔三份，第三笔两份，第四笔一份。

那么在趋势抬高的过程中，你的平均持仓位的价格就比较低，也就是过去老话讲的，要信的早信。

但实际上，真到了操作时，明明懂这句话，依然会做反过来，也就是前轻后重。

第一笔下单一份，第二笔两份，第三笔三份，第四笔四份。

最后一看，自己的持仓位的价格是比较高的，因为大头都是涨上去才下的单。

也就是说，我没有办法做到要信早信，而是反过来了。

**它涨了我才信，它越涨我越信，以至于我的持仓位是很高的。**

同一笔交易，我承担了更高的风险，失去了更多的本该属于自己的利润。

这是为什么？

不少读者都问过类似的问题，那我们来看这个问题。

咱们在寻找真相的过程中，不要瞎猜，我们要用大数据分析，你找很多投资人样本案例过来。

在大数据统计下，什么样的人更容易采用第一种模式，什么样的人，更容易陷入第二种模式。

统计过后你就会发现，这根本是两种人，两种完全不同投资经验的群体。

第二种模式下的人，大部分都是新手，他们几乎都是刚接触，或者有过短暂的的投资经历，但也基本上属于赌运气。

而第一种模式下的人，大部分都是老手，他们都有丰富的投资经验。

你看到问题了么？

第二种模式的人，他是希望通过一搏，而第一种模式的人，他们实际上通过了两搏。

两搏也只是虚指，现实中他们可能要经历四搏。

第一搏是熟悉市场，成天看盘，了解信息，就像过去游戏厅里的孩子，没有硬币，站在别人背后看人家打。

第二搏是虚拟仓，他拿着练功券在那里大量的投资交易，去进一步提升自己对策略的熟悉程度。

第三搏是实战，他开始用一些小钱，慢慢的试图赚钱，稳定了，再逐渐投入，直到变成老手。

第四搏才轮到我们上面讲的某一次机会，他是按照四份，三份，两份，一份的原则下单

你看到区别了么？

你是听说开飞机能挣钱，直接就跑进驾驶舱，开上民航的飞机就起飞了。

他是听说开飞机能挣钱，经过了航校阶段，经过了模拟驾驶阶段，经过了上千个小时的实飞，最后，去从副机长开始飞行员的生涯。

他经过了四步，你只经过了一步。

每增加一个步骤，淘汰一半人，四个步骤之后，94%的人，就都被淘汰了。

你要知道，我打的这个比方并不准确，航校是有强迫性的，每天像上班一样，有人盯着你，要干嘛干嘛，做错了有纠正，睡懒觉被叫家长。

这个体系本身依然像打工。

你从小到大，做过的所有的超过一个步骤的体系，比如读书，比如打工，都是在类似的机制的强制下完成的。

我们想一想，一群小孩，我们不强迫他去读书，给你零食，给你游戏，任你冲浪上网。

你想干嘛都可以，你也可以像风间小朋友一样自己学知识，你也可以像小新小朋友一样看小姐姐。

都随你们自愿，就让你们这么自愿的长到18岁，我突然拉你们去高考。

请问，还有多少人，能考上大学？

这个社会学的试验我们当年做过的，你去看下70年代末刚恢复高考的前几届，有多少人考上大学。
十年，整整十年，你不用学习，放羊，我突然开始要考试了。

这种考核才是在选拔真人才。

也就是说，没有任何外力的强加，没有任何系统的保障，你要纯粹自觉自愿的，独自一个人，穿越多个步骤。

每增加一个步骤，就会干掉一半人，这个自然规律，就会起作用。

我们今天看不到真实的数据，我们今天到高三了居然还有一半人能考上大学，那是人为扭曲过的数据。

这个人，他本不可能独自穿越多个步骤，结果，找了一堆的老师，家长，硬是填鸭填过关了。

那我们来想象下，现在他毕业了，他赚不到钱，你又埋怨他，这不是你自己双标么？

他本来就没道理赚到钱呀。
