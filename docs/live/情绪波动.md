# 炒股的情绪波动
记录自己买卖股票时的情绪波动，希望有助于以后的炒股。目前发现以下几点:
```markdown
1. 股票在上涨过程中，会忍不住一直查看自己涨了多少钱
2. 在股票下跌过程中，基本不会看
3. 有盈利并且盈利较多时，尽管看着股票趋势是下跌的，会去幻想这是一个小回调，**没有及时止盈**
4. 目前发现止盈止损都很难受，但是止盈的难受程度大于止损。
```

需要学习的地方:
```markdown
1. 控制情绪，发现自己情绪波动比较大时，将影响自己情绪的股票进行清仓处理，如果还不能平复情绪，全部清仓
2. 学会控制情绪。
```

## 2023 - 6月份
### 06-14 星期三
早上预估工商银行会下跌，开盘时看到股市一片红没有在意，隔一段时间后看到工商下跌1个点，盈利又往回吐后心态不稳，对工商银行清仓处理。

### 06-15 星期四
1. 京东方股票一直处于不动的状态，看MACD已经处于上涨中期，处理了二分之一的股票
2. 根据双针探底的策略选了嘉事堂
3. 根据MACD的走势选了富春环保
目前自己选股的策略是看MACD的走势，MACD是根据股票的走势行程的，也就是说我还是靠猜测选股。目前没有更好的策略，就是看MACD，猜测后期的股票走势。

>心态很容易受到股票走势波动，走势好的时候想要加仓，走势坏的时候想直接清仓掉，如果有盈利回吐想着会不会后续会再涨回来。

### 06-16 星期五
1. 昨天使用双针探底 + MACD 选择的嘉事堂，阴跌，太影响情绪了直接卖掉了。

> 学会清零心态，买过了不要关注钱赚了还是亏了，保持清零心态，只看走势。
